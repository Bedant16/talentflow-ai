# TalentFlow AI

TalentFlow AI is an automated resume screening workflow built using n8n.

## Features
- Monitors Gmail inbox for new applications
- Extracts PDF resumes automatically
- Uses AI to score candidates
- Shortlists top applicants
- Logs data into Google Sheets

## Tech Stack
- n8n
- OpenAI API
- Gmail API
- Google Sheets

## Files
- workflow.json
- screenshots/

## How to Run
1. Import workflow.json into n8n
2. Add credentials
3. Run workflow
